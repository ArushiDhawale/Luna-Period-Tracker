# Hustlers
Period tracker 

Overview
A comprehensive web application designed to track menstrual cycles and provide health insights based on the user's current cycle phase. This tool goes beyond simple date tracking by integrating lifestyle management features such as phase-specific diet plans, workout routines, and medication adherence tools.

Key Features
1. Cycle Tracking & Logging
Users can log start and end dates of their periods.
Symptom and mood logging capabilities for historical analysis.

2. Predictive Algorithms
Calculates and predicts the start date of the next menstrual cycle.
Estimates fertility windows and ovulation days based on historical data.

3. Phase-Based Recommendations
Diet Plan: customized nutritional suggestions tailored to the specific phase (Menstrual, Follicular, Ovulation, Luteal).
Custom Routines: Workout and productivity suggestions aligned with hormonal energy levels.

4. Medication Reminders
Notification system for daily contraceptives, supplements, or pain management medication.
